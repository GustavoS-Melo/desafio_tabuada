> Repositório arquivado — projeto de estudo inicial, mantido como histórico de aprendizado.


Este projeto foi criado para praticar JavaScript, especialmente lógica de programação, manipulação do DOM, validações e eventos.

O projeto inclui validações inteligentes de peso e altura, mensagens claras, cores dinâmicas baseadas no resultado e uma interface responsiva.

Funcionalidades:

Calcular automaticamente o IMC do usuário
Exibir a classificação correspondente (abaixo do peso, normal, sobrepeso, obesidade…)
Validação de peso mínimo/máximo
Validação de altura mínima/máxima
Exibição de mensagens de erro
Resultado com fundo colorido, variando conforme o nível do IMC
Tabela explicativa das classificações no próprio HTML
Layout responsivo (via media queries)



Tecnologias utilizadas:


HTML5

CSS3

Media Queries para design responsivo

JavaScript
